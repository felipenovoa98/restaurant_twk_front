dentro de script 
Consejo: También puede utilizar el siguiente javascript
 para cerrar el modal haciendo clic fuera del contenido modal (y no solo usando el botón "x" o "cancelar" para cerrarlo):


<script>
// Obtener el modal
var modal = document.getElementById('id01');

//Cuando el usuario haga clic en cualquier lugar fuera del modal, ciérrelo.
window.onclick = function(event) {
  if (event.target == modal) {
    modal.style.display = "none";
  }
}
</script>
